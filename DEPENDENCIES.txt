================================================================================
HackApp - Complete Dependency List
================================================================================

This file lists ALL dependencies required for the HackApp project.

================================================================================
SYSTEM DEPENDENCIES (Install with apt-get or system package manager)
================================================================================

Required for audio recording (sounddevice):
  sudo apt-get install portaudio19-dev

Required for Python development (if not already installed):
  sudo apt-get install python3-dev python3-pip

Optional - For better audio quality:
  sudo apt-get install libportaudio2 libportaudiocpp0

================================================================================
PYTHON DEPENDENCIES (Install with pip)
================================================================================

You can install all Python dependencies with:
  pip install -r hackapp/agent/requirements.txt
  pip install -r hackapp/middleware/requirements.txt
  pip install -r hackapp/mock_service/requirements.txt

Or install individually:

================================================================================
QUICK INSTALL COMMAND
================================================================================

Full installation on Ubuntu/Debian WSL:

# System dependencies
sudo apt-get update
sudo apt-get install -y portaudio19-dev python3-dev python3-pip

# Python dependencies (with --break-system-packages for WSL)
cd /path/to/Hackathon_HDF
pip install -r hackapp/agent/requirements.txt --break-system-packages
pip install -r hackapp/middleware/requirements.txt --break-system-packages
pip install -r hackapp/mock_service/requirements.txt --break-system-packages

# Optional: OCR and Excel support
pip install pytesseract pillow openpyxl pandas --break-system-packages

================================================================================
TROUBLESHOOTING
================================================================================

1. If pynput fails:
   - WSL users: You may need to run agent on Windows directly
   - Install: pip install python-xlib (Linux)

2. If sounddevice fails:
   - Install: sudo apt-get install portaudio19-dev
   - Then: pip install sounddevice --force-reinstall

3. If "No module named 'middleware'" error:
   - Always run from hackapp/ directory
   - Set: export PYTHONPATH=/path/to/hackapp

4. If network errors with Groq API:
   - Check internet connectivity
   - WSL: May need to configure DNS in /etc/resolv.conf
   - Try: curl -v https://api.groq.com

5. If "python-multipart" error:
   - pip install python-multipart --break-system-packages
